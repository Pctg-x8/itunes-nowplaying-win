itunes-win
=======================
[![npm version](https://badge.fury.io/js/itunes-win.svg)](https://badge.fury.io/js/itunes-win)

Control iTunes app for Windows.

## Requirement

- Windows
- Node.js
- [node-gyp](https://github.com/nodejs/node-gyp)

## Configure

```
$ node-gyp configure
```

## API

### `getCurrentTrack()`
Get current playing track info.
